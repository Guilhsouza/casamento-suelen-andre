# 💍 Casamento Suelen e André

Este projeto foi criado como um presente de casamento para meu casal de amigos a **Suelen** e o **André**.  
A proposta foi desenvolver uma experiência digital personalizada, elegante e emocionante, celebrando esse momento tão especial.

## ✨ Sobre o Projeto

O site foi desenvolvido com foco em simplicidade, leveza e animações suaves, oferecendo uma navegação agradável e visual moderno.  
Todo o layout, tipografia e interações foram pensados para transmitir carinho, alegria e delicadeza — tudo que um casamento merece.
